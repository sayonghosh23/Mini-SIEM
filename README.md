# Mini SIEM Dashboard

A Security Information and Event Management (SIEM) dashboard built with Python and FastAPI.

## Features

- Log Collection
- Alert Generation
- Event Correlation
- Dashboard Visualization

## Tech Stack

- Python
- FastAPI
- SQLite
- React


## Current Features

- Log Ingestion API
- SQLite Database Storage
- Threat Detection Engine
- Alert Generation
- Log Search
- Attack Simulation
